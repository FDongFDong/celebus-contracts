📱 실제 프로덕션 환경:

  • User 1의 기기: 질문 1개 답변 → 서명 → 백엔드 전송
  • User 2의 기기: 같은 질문 답변 → 서명 → 백엔드 전송
  • 백엔드 서버: 개별 서명 수집 → 배치 결합 → Executor 서명 → 컨트랙트 제출

💡 이 데모에서는 편의상 한 페이지에서 모든 역할을 시뮬레이션합니다
