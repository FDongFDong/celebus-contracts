MainVoting 데모

EIP-712 서명 기반 배치 투표 시스템

이 데모는 프론트엔드 통합 테스트용입니다

실제 프로덕션 환경:

  • User 1의 기기: User 1 레코드 생성 → User 1 서명 → 백엔드 전송
  • User 2의 기기: User 2 레코드 생성 → User 2 서명 → 백엔드 전송
  • 백엔드 서버: 여러 사용자 레코드 수집 → 배치 결합 → Executor 서명 → 컨트랙트 제출

이 데모에서는 편의상 한 페이지에서 모든 역할을 시뮬레이션합니다
